一、接口总述
接口地址： http://url.cqcn/api
返回格式： json
请求方式： get
请求示例： http://url.cq.cn/api?url=http://url.cq.cn
二、请求参数
请求参数说明：
参数 类型 必须 说明 示例
id int true ID id=001
key string true PAI KEY key=sdjfbsdagjgba
url string true 请求的url url= api.mo.cn

三、返回参数
返回参数说明：
参数 类型 示例 说明
success 布尔 ture 成功
content json {"url":"http:\/\/url.cq.cn\/9966"} 返回内容
    url string http:\/\/url.cq.cn\/9966 返回的url
四、返回数据
 
{
"success":"ture",
"content":{
    "url":"http:\/\/url.cq.cn\/9966"
}
}
